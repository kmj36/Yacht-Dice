# 설명
주사위 게임 요트다이스를 C언어로 개발하였습니다.
소스파일의 확장자는 C++이지만 C언어의 함수만을 사용하였습니다.

유튜브의 무료 음원 2개
1. Kevin MacLeod - Samba Isobel
2. Kevin MacLeod - Apero Hour
를 사용하여 브금을 넣었고,

규칙은 나무위키의 요트(게임)란에 51 Worldwide Games 점수 계산법을 인용하였습니다.
